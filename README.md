# Weather App Interface

A beautiful and responsive weather application interface built with HTML and CSS.

## Features
- **Weather Explorer**: Search for a city to get the latest weather updates.
- **Current Weather Info**: Displays the temperature and weather conditions (e.g., Clear Sky) for the searched city.
- **4-Day Forecast**: Shows a prediction of the weather for the upcoming days.
- **Attractive UI**: Features a clean and modern design with hover animations, Font Awesome icons, and a visually appealing background.

## Technologies Used
- HTML5
- CSS3 (Flexbox, custom styling, UI animations)
- [Font Awesome](https://fontawesome.com/) (For weather and UI icons)

## Project Structure
- `index.html`: The main HTML document containing the structure of the weather app interface.
- `style.css`: The stylesheet containing the designs, layouts, and animations.
- `thunderstorm-countryside.jpg`: Background image used in the application.

## How to Run Locally
1. Clone or download the project files.
2. Ensure you have the files `index.html`, `style.css`, and `thunderstorm-countryside.jpg` in the same directory.
3. Open `index.html` in any modern web browser to view the interface.
